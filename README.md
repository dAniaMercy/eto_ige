## Задача №22347

S = 2322000 * 1024 * 8  
s = S / 120  

Vo = 3840 * 2160 * 20  

Vc = 1280 * 720 * b  

Vo - Vc = s  

b = (Vo - s) / (1280 * 720) = 8  

2^8 = 256  

**Ответ: 256**
